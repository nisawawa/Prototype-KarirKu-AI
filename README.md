# KarirKu AI - Prototype

**KarirKu AI** adalah sebuah purwarupa (prototype) platform pengembangan karir berbasis kecerdasan buatan (AI) yang dirancang untuk membantu mahasiswa, lulusan baru (fresh graduates), dan profesional muda dalam menavigasi perjalanan karir mereka. 

## 🚀 Fitur Utama

- **Dashboard & Analitik**: Pantau perkembangan profil, skor kecocokan karir, dan aktivitas terbaru Anda.
- **AI Job Matcher**: Sistem pencocokan cerdas yang merekomendasikan lowongan pekerjaan berdasarkan keahlian, pengalaman, dan minat Anda.
- **Mock Interview Berbasis AI**: Latih kemampuan wawancara Anda melalui simulasi interaktif dengan AI.
- **Interview Feedback & Analisis**: Dapatkan umpan balik yang komprehensif setelah melakukan simulasi wawancara untuk terus berkembang.
- **Manajemen Profil Terpusat**: Bangun profil profesional Anda dengan mudah.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web modern:
- **[React](https://react.dev/)** (v19) - Library UI
- **[Vite](https://vitejs.dev/)** - Frontend tooling & bundler
- **[Tailwind CSS](https://tailwindcss.com/)** (v4) - Utility-first CSS framework untuk styling
- **[Framer Motion](https://motion.dev/)** - Animasi interaktif yang mulus
- **[Lucide React](https://lucide.dev/)** - Kumpulan ikon SVG modern
- **[React Router](https://reactrouter.com/)** - Manajemen navigasi (Routing)

## 📦 Instalasi & Menjalankan Proyek Secara Lokal

Pastikan Anda telah menginstal **Node.js** di sistem Anda.

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/nisawawa/Prototype-KarirKu-AI.git
   ```

2. **Masuk ke direktori proyek:**
   ```bash
   cd Prototype-KarirKu-AI
   ```

3. **Install dependensi:**
   ```bash
   npm install
   ```

4. **Jalankan server pengembangan (development server):**
   ```bash
   npm run dev
   ```
   Aplikasi akan berjalan di `http://localhost:5173/` (bisa bervariasi jika port sudah terpakai).

## 📂 Struktur Folder Utama

- `/src/pages` - Berisi halaman-halaman utama seperti Landing Page, Dashboard, Job Matcher, Mock Interview, dll.
- `/src/layouts` - Layout struktur aplikasi (misal: Navbar dan Sidebar untuk halaman Dashboard).
- `/src/components` - Komponen UI pendukung yang dapat digunakan kembali.
- `/src/data` - Data statis (mock data) untuk keperluan prototipe aplikasi.

## 👥 Tentang Proyek

Proyek ini merupakan purwarupa (prototype) awal yang dibangun sebagai bagian dari Tugas Besar Kewirausahaan (TUBES KWU).
